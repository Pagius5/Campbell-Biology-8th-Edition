When you do chromatographic separations for different lengths of time, the pigments migrate a different distance on each run. However, the migration of each pigment relative to the migration of the solvent does not change. This migration of pigment relative to migration of solvent is expressed as a constant, Rf (Reference front). It can be used to identify a specific pigment and is calculated by using the formula on the left.

So for example, if the solvent traveled 8 cm and pigment X traveled 6 cm, then the Rf can be calculated by dividing 6 cm by 8 cm. The Rf value would be 0.75.
