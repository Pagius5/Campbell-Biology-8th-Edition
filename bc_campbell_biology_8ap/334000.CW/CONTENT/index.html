Welcome to the Student Media CD-ROM to accompany BIOLOGY 8e AP Edition. This CD contains most of the information and resources found on the BIOLOGY 8e AP Edition website. Both the CD and the website are specially designed resources for Advanced Placement Biology students and teachers.

To begin, select a chapter from the chapter menu above and click Go.

On this CD you will find interactive tutorials, videos, online labs, graphing exercises, and several forms of assessment. Although this CD-ROM contains most of the assets and activities from the website, there are a few additional resources found only on the site. Those are: the gradebook function for teachers, an electronic version of the textbook (eBook), art and photos from the textbook, website links (including AP-specific links), videos, BioFlix animations, MP3 audio tutorials, cumulative test builder, and special tools for AP teachers.

Students' access to the website is available through their teacher. Instructor access is provided upon textbook adoption. Textbook shipments should include an access card which contains registration instructions and codes. A single set of website access codes can be used by two teachers and up to 50 students for one year. If for some reason the access card is not included with the first textbook shipment, or an instructor inherits a set of texts, new access codes can be requested by emailing phwebaccess@pearsoned.com. In subsequent out-years of the adoption, registered teachers will be sent an email in the summer with information for setting up website access for a new set of students.
