I. HOW TO START THE CD-ROM
PC Users:
1. Please close all browser windows before inserting this CD 
2. Insert the CD-ROM into computer
3. Click on "Launch Campbell Biology" (If browser
window does not launch automatically, run "StartHere.exe" found on the CD.)

MAC Users:
1. Please close all browser windows before inserting this CD
2. Insert the CD-ROM into computer
3. Double-click on the CD-ROM image on your desktop
4. Click "StartHere.html"
5. Click on "Launch Campbell Biology"
